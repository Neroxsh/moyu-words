## 安装

| 平台 | 文件 |
| :--- | :--- |
| macOS 11+ | `摸鱼背词_*.dmg` |
| Windows 10+ | `MoyuWords_*_setup.exe` |

> macOS 首次打开如提示"无法验证开发者"，前往 **系统设置 → 隐私与安全性 → 仍要打开**。
>
> Windows 安装包暂未签名，Smartscreen 可能提示风险，点击"更多信息 → 仍要运行"即可。

## 功能

- **摸鱼窗口** — 透明、置顶、可拖拽缩放。路人视角：你在工作
- **智能复习** — 鼠标左右键切词，长按标记熟识，自动归档
- **碎裂动画** — 搞定一个单词，就让它碎掉
- **学习计划** — 选书 → 定天数 → 自动拆为每日单元
- **进度管理** — 单元打卡、熟识词库、一键导出 CSV
- **词书自由** — 内置 7 本 + 支持 `.txt` `.csv` `.json` 导入
